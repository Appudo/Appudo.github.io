  • printSub()

    printSub is part of the template engine for pages. To walk and output a subpart the function is called as often as the subpart should be repeated. Each subpart has its own function from within printSub must be called. With the call to printSub the subpart is parsed and nested subparts and markers are executed.

    Declaration

    Swift

    public func printSub()
  • readSub()

    readSub is part of the template engine for pages. With the call to printSub the subpart is parsed and nested subparts and markers are executed. The result is not added to the output of the page but returned as a String.
