Account

public struct Account

Single accounts are added, updated and removed with an Account. Appudo is a multi account system where each account has multiple users. An Account can have multiple domains to access its runs. To use this API the user of the current login must be in the master group.

  • swap(_:)

    Switch to another account . Members of the master group can do a temporary switch to another account for e.g. file operations. Switch back to the initial account with AccountID(-1).

    The only shared group between accounts is the master group. A file or directory from another account must be accessible by the master group or all others to work with it.
