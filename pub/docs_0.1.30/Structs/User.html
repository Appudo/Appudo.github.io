User

public struct User

Single users are added, modified or removed with a User. Other APIs depend on the current login user and it’s group membership. Also files and other items are owned by users.

  • update(_:)

    Update the current login user before the login expires. Logins can be set to automatically expire after a time period.

    Declaration

    Swift

    static public func update(_ expire : Int) -> Void

    Parameters

    expire

    The time until the login expires in seconds.
  • swap(_:)

    Switch to another user. Members of the backend group can do a temporary switch to another user if they own it. Switch back to the initial login user with UserID(-1). If the new user is a member of the master group the current user also must be a member. A switch to the initial owner of the current run is always possible. Be aware that a run owned by a member of e.g. the admin group is a security risk and should be avoided if not absolutely necessary.

    Declaration

    Swift

    static public func swap(_ user : UserID = UserID(-1)) -> AsyncBool

    Parameters

    user

    The user to switch to.
